TailwindCSS Version Check

Diese Seite hilft Ihnen zu überprüfen, ob die TailwindCSS-Konfiguration korrekt ist und zeigt Ihnen, wie Sie auf Version 4.1.10 aktualisieren können.

Bereit zum Ausführen...

Manuelle Überprüfung

  1. Öffnen Sie die terminal in VSCode
  2. Führen Sie npm list tailwindcss aus, um die installierte Version zu prüfen
  3. Führen Sie code --list-extensions | findstr tailwind aus, um die Extension zu prüfen
